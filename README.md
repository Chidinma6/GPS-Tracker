# GPS-Tracker
This GPS tracker prototype was programmed using C++ on the Arduino IDE. The main components used were an ESP32 microcontroller and a GPS module.

For a more compact and practical prototype, the system can be designed using an ESP32, GPS module, SIM module, and a battery.

The ESP32 serves as the microcontroller and was chosen because of its built-in Wi-Fi capabilities, processing power, and low power consumption. The GPS module is responsible for collecting and transmitting location data. A battery provides portable power to the system, while connecting wires are used to establish communication between the components. The addition of a SIM module enables the device to transmit location information over cellular networks, allowing for real-time tracking even when Wi-Fi is unavailable.
